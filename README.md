## How to Use ðŸ“‹

1. **Clone the Repository**:
    ```sh
    git clone https://github.com/yourusername/fresher-intro-tool.git
    ```
2. **Open the HTML File**:
    Open `index.html` in your preferred web browser.

3. **Fill the Form**:
